# Staj Durağı

Üniversite öğrencilerinin staj yeri bulmalarına yardımcı olan modern web platformu.

## Proje Hakkında

Bu proje, TÜBİTAK-2209-A Üniversite Öğrencileri Araştırma Projeleri Desteği Programı kapsamında, İstanbul Gedik Üniversitesi öğrencileri tarafından geliştirilmiştir.

## Özellikler

- **Öğrenci Paneli**: Profil yönetimi, özgeçmiş, başvuru takibi, staj ilanları
- **Firma Paneli**: İlan yayımlama, profil yönetimi, başvuru inceleme
- **Duyurular**: Kariyer etkinlikleri ve önemli duyurular
- **Modern UI**: Kullanıcı dostu ve ergonomik arayüz

## Teknolojiler

- React.js 18
- React Router v6
- Vite
- Modern CSS

## Kurulum

```bash
npm install
npm run dev
```
